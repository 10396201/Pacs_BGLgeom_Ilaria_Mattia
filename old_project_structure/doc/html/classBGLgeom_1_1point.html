Detailed Description

template<unsigned int dim, typename Storage_t = double>
class BGLgeom::point< dim, Storage_t >

Class template for storing the vertex coordinates in n-dimentional space.

Note:
Constructors and set method are implemented with std::initializer_list, so they have to be called with: method({args})
Parameters:
dim Template argument that specifies the dimension of the space
Storage_t Template argument that specifies the precision type for the coordinates
